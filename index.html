隱私權政策 Privacy Policy

最後更新：2026 年 2 月 24 日

一、關於我們

威達興業有限公司（Dynamax Petrochemical Ltd.，以下簡稱「本公司」）為泰國國家石油 PTT Lubricants 在台灣之授權經銷商，營運「PTT Lube Taiwan」Facebook 粉絲專頁。

二、資料蒐集範圍

本公司透過 Facebook Graph API 蒐集之資料僅限於：

  • PTT Lubricants Thailand 及 PTT Lubricants Motor Sport 公開粉絲專頁之公開貼文內容（文字、圖片、發布時間、貼文連結）
  • 本公司管理之 PTT Lube Taiwan 粉絲專頁相關數據

我們不蒐集：個人用戶資料、私人訊息內容、留言者個人資訊、粉絲名單或任何非公開資訊。

三、資料使用目的

所蒐集之公開貼文資料僅用於：

  • 將泰國總公司之行銷內容翻譯為繁體中文
  • 在地化調整後發布至台灣粉絲專頁
  • 內部內容管理與排程

資料不會用於廣告投放、用戶分析、轉售或任何其他商業用途。

四、資料儲存與安全

所有資料儲存於本公司私有伺服器，採取以下安全措施：

  • API 金鑰與存取權杖以加密方式儲存
  • 僅授權人員可存取內容管理系統
  • 已處理之來源資料於 90 日內清除

五、資料分享

本公司不會將任何 Facebook 資料出售、出租、授權或分享予第三方。翻譯過程中使用之 AI 翻譯服務僅作為暫時性輸入處理，不會保留任何 Facebook 資料。

六、您的權利

如您對本公司之資料處理有任何疑問或請求，歡迎透過以下方式聯繫：

  • 公司名稱：威達興業有限公司
  • 電子郵件：benson.pcl@gmail.com
  • Facebook：PTT Lube Taiwan

七、法規遵循

本公司遵守 Meta 平台條款、Facebook 開發者政策，以及中華民國個人資料保護法之相關規定。我們僅透過官方 Graph API 端點存取 Facebook 資料，且不超出授權權限之範圍。

八、政策更新

本隱私權政策如有修訂，將於本頁面公告最新版本。建議您定期查閱。
